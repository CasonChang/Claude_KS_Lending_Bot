# 學習檢討 2026-07-06（UTC 日）· Day 5 —— 對方的 13% 神話破功

## 破案：那筆「比較低的掛單」是怎麼來的（8.33%）

使用者觀察到子帳戶掛了一筆明顯低的單——查證後真相很清楚，而且很重要：

**對方 07-04 鎖的 13.02% 是 2 天單，07-06 陸續到期。偏偏到期時撞上市場崩盤**
（當日錨點一度殺到 6.3%、隊首最低掛單價 best-ask 只剩 5–8%、FRR 雖仍釘 14.48 但那是遲滯值）。
他們「利用率優先、掛低求成交」的核心哲學這時原形畢露：

```
08:00  一批 13% 到期 → 立刻掛 8.33% → 08:10 秒成交（市場錨點才 6.3）
11:50  又一批到期 → 先試掛 11.1–12.5、10 分沒中 → 撤掉改掛 8.33–9.64 → 秒成交
12:10  市場回一點 → 改掛 11.63 → 成交
整日 book 加權：13.02% → 12.79 → 12.2 → 11.83 → 11.64 → 11.46（一路被稀釋）
```

**結論：對方的 13% 不是可持續的實力，是 07-04 那個多 spike 日的好進場點，鎖了 2 天。
一旦到期、又碰上軟盤，他們比我們更狠地往下追（直接掛 8.33% 求成交）。**

## 核心發現：崩盤日兩邊都逃不過 book 陰跌，只是姿勢不同

- **對方**：利用率優先 → 到期就算掛 8.33% 也要成交，book 被自己往下拖得比我們兇；
  好處是幾乎不閒置（83%，空窗只在重定價那幾分鐘）。
- **我們**：稍微沉得住氣（新成交 9.44% > 對方部分 8.33%），利用率反而更高（97%）、
  現金當日還多賺——**軟盤這天我們的鎖倉 book＋不追到骨折，表現不輸甚至略好。**
- 昨天我擔心的「我們 book 陰跌」今天看，對方其實**跌得更快**（13→11.46 一天內），
  因為他們全 2 天、每次到期都得面對當下市場；我們的 30 天鎖倉反而是**抗跌的錨**。
